# 接口规范

## 接口格式

### 对于一些 GET 请求

```js
// GET /api/v1/

{
  sender: `${user}@${platform}`.toBase64(),
  ...params
}
```

## 错误代码

### 未知错误

| 错误代码 | 错误说明 | 状态码 | 备注 |
| -------- | -------- | ------ | ---- |
| 0        | 未知错误 | 500    |      |

### 请求错误

API 错误代码中 scope = 'api'

| 错误代码 | 错误说明   | 状态码 | 备注 |
| -------- | ---------- | ------ | ---- |
| 1001     | 无来源请求 | 401    |      |
| 1002     | 不合法参数 | 400    |      |
